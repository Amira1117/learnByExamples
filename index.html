Изучаем html/css/js на примерах.

Для начинающих.

Нужно установить и настроить Visual Studio Code (VSC)

Руководство

Руководство

Необходимо создать папку, например под именем projects. Создать в нем папку например myFirstProject. Открыть эту папку из VSC меню File/Open folder...

Создать первый файл в проекте, например назвать index.html. Используя шаблон создать первый сайт, для этого внутри файла index.html нужно набрать "!"+enter. Внутри тега body написеть "Hello World!". Теперь чтобы открыть это в браузере нужно нажать "Go Live", это внизу справа у VSC, если правильно установлен плагин "Live Server"

Нужно создать страничку в папке solutions, назвать его style.html. Добавить куплет от стихотворения "В лесу родиласть ёлочка"

Тест рассположить по центру стриницы

Внутри сайта включить стили: фон голубой, первые буквы строк стихотворения раскрашены в разные цвета

Информацию о стилях брать с сайта htmlbook или запросами goolge
